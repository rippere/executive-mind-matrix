# P3.1.3: Command Center Auto-Refresh - Quick Reference

## What It Does
Automatically refreshes Command Center metrics every 15 minutes without manual intervention.

## Configuration

### Enable/Disable
```bash
# In .env file
COMMAND_CENTER_REFRESH_ENABLED=true   # or false
```

### Change Interval
```bash
# In .env file (value in minutes)
COMMAND_CENTER_REFRESH_INTERVAL=15    # default: 15 minutes
```

## API Endpoints

### View Scheduled Jobs
```bash
GET /scheduler/jobs
```

### Manual Refresh (Testing)
```bash
POST /command-center/update-metrics
```

## Monitoring

### View Prometheus Metrics
```bash
GET /metrics

# Look for:
# - command_center_refreshes_total
# - command_center_refresh_duration_seconds
```

### View Logs
```bash
# Startup
INFO: Command Center auto-refresh started (every 15 minutes)

# Refresh
INFO: Starting scheduled Command Center metrics refresh
SUCCESS: Command Center metrics refreshed successfully in 0.23s
```

## Troubleshooting

### Not Running?
Check if P2 features are available:
```bash
GET /
# Look for: "scheduler": true
```

### Errors?
Check logs:
```bash
tail -f logs/app.log | grep "Command Center"
```

### Disable Quickly
```bash
# Set in environment
export COMMAND_CENTER_REFRESH_ENABLED=false

# Restart application
```
